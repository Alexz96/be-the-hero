# Como executar o projeto

## Back end

```bash
# clonar repositório
git clone https://github.com/Alexz96/be-the-hero.git

# entrar na pasta do projeto back end
cd backend

# instalar as dependências
npm install ou yarn install

# executar o projeto
npm start ou yarn start
```
